# Pandanus(露兜树)

> JCR(Java content repository) reference application, powered by Shunyi Chen, based on a simplified spring microservices architecture and Docker containers.

---

## 技术栈

- [x] Spring Cloud 2022.0.0

- [x] Spring Boot 3.0.1

- [x] Spring Data JPA

- [x] JDK17

- [x] Postgres

- [x] Consul

- [x] Docker

- [x] Keycloak

- [x] OAuth2

- [x] Apache Jackrabbit
